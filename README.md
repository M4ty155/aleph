Declaracion de variables: _var
Asignacion de variables: _var = {conjunto_elementos} para conjuntos
                         _var = [lista_elementos] para listas
Asignacion multiple: _var1, _var2,...,varn = elem1, elem2,..., elemn
Operaciones disponibles:
                        Conjuntos:
                                +: Union de conjuntos _A + _B
                                -: Diferencia de conjuntos _A - _B
                                *: Interseccion de conjuntos _A * _B
                        Listas:
                            push(_var, elem) //agrega un elemento a la lista
                            pop(_var) //quita el primer elemento de la lista
Sentencia if: if(condicion){lista_sentencias}
              if(condicion){lista_sentencias}else{lista_sentencias}
